🗄 Step 1 — Enable Vercel KV (Database)

1

Go to your project on vercel.com → click the Storage tab

2

Click Create Database → select KV → click Create

3

Click Connect Project → select your UCF project → Confirm. Vercel automatically adds the required environment variables.

4

Redeploy your project once (push any small change to GitHub). The database is now live.

📧 Step 2 — Set Up Email (Resend)

1

Go to resend.com → sign up for a free account (100 emails/day free)

2

Click API Keys → Create API Key → copy it

3

In Vercel → your project → Settings → Environment Variables → add:

RESEND_API_KEY
re_xxxxxxxxxxxxxxxxxxxxxxxx (your key)
4

Click Save → Redeploy. Daily digests will now be sent to cbazela@crgfinancial.com every morning at 6am ET.

Resend's free tier sends from onboarding@resend.dev. To send from digest@crgfinancial.com, add your domain in Resend's dashboard and verify a DNS record. Takes 5 minutes.

⏰ Step 3 — Cron Schedule

The daily scan is already configured in vercel.json and will run automatically at 6:00am ET every day once Vercel KV is connected. No additional setup needed.

0 11 * * * → 6:00am Eastern Time

Cron jobs require Vercel's Pro plan ($20/mo) or higher. On the free Hobby plan you can still run searches manually from this tool — just not the automated daily scan.
